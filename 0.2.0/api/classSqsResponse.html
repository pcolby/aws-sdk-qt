QString SqsResponse::errorString ( ) const
virtual

Get this response's error string.

This override checks for SQS service errors, and if found, returns the first one. Otherwise, this override delegates to the base implementation (which checks for network and XML parse errors, for example).

Also consider using the serviceErrors function to retrieve all of the servive errors (if any), not just the first one (though often SQS will only return one error per response).

Returns
An error string, or a null QString if this response has no errors.

bool SqsResponse::hasError ( ) const
virtual

Did any error occurr processing this reponse?

Errors could be anything from network requests errors, to errors validating AWS response content.

This base implementation supports network, and XML parsing errors. Derived classes may override this function to support additional error types.

Returns
true if this response contains one or more errors, false otherwise.
Note
This function may be equivalent to !isValid, however, it is not safe for this function's implementation to be just that, since derived classes may assume the inverse, resulting in an infinite cycle between the two functions.

bool SqsResponse::isValid ( ) const

Is this AWS response valid?

This base implementation simply checks for the absense of network and XML parsing errors. Derived classes may override this function to check other error types, and/or apply additional validation rules.

Returns
true if this response is considered valid, false otherwise.

void SqsResponse::parseFailure ( QIODevice &  response)
protectedvirtualslot

Parse a failure response.

This implementation parses SQS ErrorResponse elements, as defined by http://queue.amazonaws.com/doc/2012-11-05/QueueService.wsdl

For example:

1 <ErrorResponse>
2  <Error>
3  <Type>Sender</Type>
4  <Code>AccessDenied</Code>
5  <Message>Access to the resource http://sqs.us-east-1.amazonaws.com/ is denied.</Message>
6  <Detail/>
7  </Error>
8  <Error>
9  ...
10  </Error>
11  <RequestId>9a285199-c8d6-47c2-bdb2-314cb47d599d</RequestId>
12 </ErrorResponse>
Parameters
responseResponse to parse.
Note
Unlike most other SQS responses, the SQS ErrorResponse does not wrap the RequestId element in a ResponseMetadata element.
See also
http://queue.amazonaws.com/doc/2012-11-05/QueueService.wsdl
