The DeleteRegexMatchSetRequest class provides an interface for WAFRegional DeleteRegexMatchSet requests.

This is the <i>AWS WAF Regional API Reference</i> for using AWS WAF with Elastic Load Balancing (ELB) Application Load Balancers. The AWS WAF actions and data types listed in the reference are available for protecting Application Load Balancers.
