void SqsErrorPrivate::parse ( QXmlStreamReader &  xml)

Parse an SQS Error element.

This function parses Error XML elements (ie from SQS ErrorResponse elements) as specified in http://queue.amazonaws.com/doc/2012-11-05/QueueService.wsdl

For example:

1 <Error>
2  <Type>Sender</Type>
3  <Code>AccessDenied</Code>
4  <Message>Access to the resource http://sqs.us-east-1.amazonaws.com/ is denied.</Message>
5  <Detail/>
6 </Error>
Parameters
xmlXML stream to parse.
See also
http://queue.amazonaws.com/doc/2012-11-05/QueueService.wsdl
