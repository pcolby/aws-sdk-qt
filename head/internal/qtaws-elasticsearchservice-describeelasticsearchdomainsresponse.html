Detailed Description

The DescribeElasticsearchDomainsResponse class provides an interace for ElasticsearchService DescribeElasticsearchDomains responses.

<fullname>Amazon Elasticsearch Configuration Service</fullname>

Use the Amazon Elasticsearch configuration API to create, configure, and manage Elasticsearch

domains>

The endpoint for configuration service requests is region-specific: es.<i>region</i>.amazonaws.com. For example, es.us-east-1.amazonaws.com.
