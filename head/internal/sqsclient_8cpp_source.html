97 /**
98  * @brief Constructs a new SqsClient object.
99  *
100  * This overload allows the caller to specify the specific SQS endpoint to send
101  * requests to. Typically, it is easier to use the alternative constructor,
102  * which allows the caller to specify an AWS region instead, in which case this
103  * client will determine the correct SQS endpoint for the given region
104  * automatically (via AwsEndpoint::getEndpoint).
105  *
106  * @param endpoint SQS endpoint for building requests URLs.
107  * @param credentials AWS credentials to use for signing requests.
108  * @param manager Network access manager for sending requests.
109  * @param parent This object's parent.
110  *
111  * @see AwsEndpoint::getEndpoint
112  */
