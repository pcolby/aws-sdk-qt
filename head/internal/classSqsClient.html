Constructs a new SqsClient object.

This overload allows the caller to specify the specific SQS endpoint to send requests to. Typically, it is easier to use the alternative constructor, which allows the caller to specify an AWS region instead, in which case this client will determine the correct SQS endpoint for the given region automatically (via AwsEndpoint::getEndpoint).
