Detailed Description

The TranslateRequest class provides an interface for Translate requests.

Member Type Documentation

enum TranslateRequest::Action
flags TranslateRequest::Actions

This enum describes the actions that can be performed as Translate requests.

ConstantValueDescription
QtAws::Translate::TranslateRequest::TranslateTextAction0Translate TranslateText action.

The Actions type is a typedef for QFlags<Action>. It stores an OR combination of Action values.

[override virtual protected] QNetworkRequest TranslateRequest::unsignedRequest(const QUrl &endpoint) const

Reimplemented from AwsAbstractRequest::unsignedRequest().

Returns a network request for the Translate request using the given endpoint.

This Translate implementation builds request URLs by combining the common query parameters (such as Action and Version), with any that have been added (via setParameter) by child classes.

TranslateRequest &TranslateRequest::operator=(const TranslateRequest &other)

[virtual] bool TranslateRequest::operator==(const TranslateRequest &other) const

Returns true if this request is equal to other; false otherwise.

Note, most derived *Request classes do not need to provider their own implementations of this function, since most such request classes rely on this class' parameters functionality for all request parameters, and that parameters map is already checked via this implementation.
