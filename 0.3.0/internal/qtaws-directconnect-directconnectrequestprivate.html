DirectConnectRequestPrivate::DirectConnectRequestPrivate(const DirectConnectRequest::Action action, DirectConnectRequest * const q)

Constructs a DirectConnectRequestPrivate object for DirectConnect action, with public implementation q.

DirectConnectRequestPrivate::DirectConnectRequestPrivate(const DirectConnectRequestPrivate &other, DirectConnectRequest * const q)

Constructs a copy of other, with public implementation q.

This copy-like constructor copies everything from other, except for the the object's pointer to its public instance - for that, q is used instead.

This is required to support the DirectConnectRequest class's copy constructor.

[static] QString DirectConnectRequestPrivate::toString(const DirectConnectRequest::Action &action)

Returns a string represention of action, or a null string if action is invalid.

This function converts DirectConnectRequest::Action enumerator values to their respective string representations, appropriate for use with the DirectConnect service's Action query parameters.
