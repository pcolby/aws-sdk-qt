Detailed Description

The CloudSearchDomainRequest class provides an interface for CloudSearchDomain requests.

Member Type Documentation

enum CloudSearchDomainRequest::Action
flags CloudSearchDomainRequest::Actions

This enum describes the actions that can be performed as CloudSearchDomain requests.

ConstantValueDescription
QtAws::CloudSearchDomain::CloudSearchDomainRequest::SearchAction0CloudSearchDomain Search action.
QtAws::CloudSearchDomain::CloudSearchDomainRequest::SuggestAction1CloudSearchDomain Suggest action.
QtAws::CloudSearchDomain::CloudSearchDomainRequest::UploadDocumentsAction2CloudSearchDomain UploadDocuments action.

The Actions type is a typedef for QFlags<Action>. It stores an OR combination of Action values.

[override virtual protected] QNetworkRequest CloudSearchDomainRequest::unsignedRequest(const QUrl &endpoint) const

Reimplemented from AwsAbstractRequest::unsignedRequest().

Returns a network request for the CloudSearchDomain request using the given endpoint.

This CloudSearchDomain implementation builds request URLs by combining the common query parameters (such as Action and Version), with any that have been added (via setParameter) by child classes.
