Detailed Description

The DynamoDBStreamsRequest class provides an interface for DynamoDBStreams requests.

Member Type Documentation

enum DynamoDBStreamsRequest::Action
flags DynamoDBStreamsRequest::Actions

This enum describes the actions that can be performed as DynamoDBStreams requests.

ConstantValueDescription
QtAws::DynamoDBStreams::DynamoDBStreamsRequest::DescribeStreamAction0DynamoDBStreams DescribeStream action.
QtAws::DynamoDBStreams::DynamoDBStreamsRequest::GetRecordsAction1DynamoDBStreams GetRecords action.
QtAws::DynamoDBStreams::DynamoDBStreamsRequest::GetShardIteratorAction2DynamoDBStreams GetShardIterator action.
QtAws::DynamoDBStreams::DynamoDBStreamsRequest::ListStreamsAction3DynamoDBStreams ListStreams action.

The Actions type is a typedef for QFlags<Action>. It stores an OR combination of Action values.

[override virtual protected] QNetworkRequest DynamoDBStreamsRequest::unsignedRequest(const QUrl &endpoint) const

Reimplemented from AwsAbstractRequest::unsignedRequest().

Returns a network request for the DynamoDBStreams request using the given endpoint.

This DynamoDBStreams implementation builds request URLs by combining the common query parameters (such as Action and Version), with any that have been added (via setParameter) by child classes.
